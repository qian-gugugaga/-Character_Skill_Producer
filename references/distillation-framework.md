# 行为蒸馏方法论

> 从角色信息到可运行的行为程序，核心方法论

## 一、行为模式的识别标准

一个观察要被认定为「核心行为模式」而非「偶尔一次的行为」，必须通过二重验证：

### 验证1: 跨场景复现
同一行为模式出现在角色经历的至少 2 个不同场景。

例：高松灯
- 场景 A：爱音邀请她组乐队 → 沉默很久后说「一辈子」
- 场景 B：在天台追回爱音 → 气喘吁吁说「都是灯的错」
→ 在人际关系关键时刻，用极其笨拙但极端真挚的方式表达需求 → 核心行为模式

### 验证2: 可执行性
能据此推断角色对新情境的可能反应。

例：如果灯被一个新朋友邀请去参加活动——
- 她会先犹豫（默认退缩）
- 如果对方表现出真正的需要，她会答应但提出一个过重的承诺（"每周都来吗？"）
→ 能推断新反应 → 这是可以写入 Skill 的行为规则

**如果只通过 1 重** → 降级为「偶尔行为」，仍可纳入但标注为低置信度。
**如果 0 重** → 可能只是特定场景的一次性反应，不纳入。

## 二、表达质感的量化方法

### 2.1 句式指纹

从角色的台词中随机抽取 20 句，统计：

| 维度 | 测量方式 |
|------|---------|
| 平均句长 | 字数/句数 |
| 疑问句比例 | 疑问句数/总句数 |
| 句中停顿频率 | 省略号、破折号出现频率 |
| 感叹句比例 | 感叹句/总句数 |
| 敬语密度 | 敬语标记/句数 |

### 2.2 风格标签

```
话多 ←→ 话少
长句 ←→ 短句/碎片
直接 ←→ 迂回
热情 ←→ 冷淡
敬语 ←→ 简体
稳定 ←→ 情绪起伏大
逻辑 ←→ 直觉/跳跃
```

### 2.3 语言标志（日文角色特化）

- **自称词**：私/僕/俺/あたし/わし/自分/うち...
- **对称词**：あなた/君/お前/てめえ/あんた/名字+さん/名字+くん...
- **语尾**：～ですわ/～だよ/～なのだ/～にゃ/～っす/～ござる...
- **口癖**：重复出现的特定短语
- **禁忌词/不会用的词**：从不使用的表达方式

## 三、矛盾处理原则

矛盾是角色的核心特征，不是需要修复的 Bug。

### 三种矛盾类型

1. **情境性矛盾**（不同场景不同反应）
   - 对 A 类型的人温柔，对 B 类型的人冷酷
   - 处理：分情境记录，不强求统一
   - 这正是行为蒸馏最重要的部分

2. **发展性矛盾**（角色成长）
   - 早期反应 A，后期反应 B
   - 处理：标注阶段，以最新状态为主但记录演变

3. **本质性张力**（内在冲突）
   - 既渴望 X 又害怕 X
   - 处理：明确记录为「核心矛盾」
   - 这通常是角色最有魅力的部分

### 错误处理方式
- ❌ 选一边忽略另一边
- ❌ 编一个调和的解释
- ❌ 假装矛盾不存在

## 四、信息不足时的处理

| 情况 | 处理 |
|------|------|
| 某个维度公开信息很少 | 标注「信息不足，此维度为推测」 |
| 只有粉丝/同人信息 | 降低置信度，标注为「同人推测」 |
| 信息相互矛盾且无法判断 | 并列呈现，让用户自行判断 |
| 角色在作品中刻意保持神秘 | 保留神秘，在 Skill 中注明「此方面原作未揭示」 |

---

## 五、角色 Skill vs 真人 Skill 的区别

| 维度 | 角色 Skill（CSP） | 真人思维 Skill |
|------|-------------------|------------------------|
| 核心 | 行为模式 + 表达质感 | 心智模型 + 决策启发式 |
| 动机 | 角色在故事中的驱动力 | 真人的价值观与目标 |
| 说话方式 | 模仿角色声音/节奏/口癖 | 模仿真人的表达DNA |
| 矛盾处理 | 保留角色内在冲突 | 保留真人观点演变与矛盾 |
| 验证方式 | 对比原作中的场景表现 | 对比真人已知立场 |
| 使用场景 | 角色扮演/对话 | 思维顾问/决策参考 |

## 六、质量自检清单

生成 Skill 后，用以下问题自检：

### 行为动态
- [ ] 每个行为模式都有至少 2 个不同场景的证据？
- [ ] 核心行为模式数在 3-7 个之间？
- [ ] 每个模式描述了触发条件（什么情况下）？
- [ ] 矛盾被保留而非消解？

### 表达质感
- [ ] 读起来有角色辨识度，不像通用 AI？
- [ ] 没有过度模仿变成 caricature？
- [ ] 语言标志（自称/语尾/口癖）准确？

### 硬约束
- [ ] 明确写了绝对不会做的事？
- [ ] 标注了信息源和调研时间？
- [ ] 承认了信息不足的维度？

### 整体
- [ ] 用这个角色的眼睛看一个新情境，反应合理可信？
- [ ] 不是原作台词的拼凑，而是行为模式的运转？
- [ ] 删掉名字和作品名后，还能认出这是哪个角色？
