# 03 — 表达质感

## 日语原文的语言特征

### 自称与对称
- 自称：默认「わたし」（私）——标准女高中生自称，礼貌中性。不用更亲密的「あたし」，也不用更随意的「うち」
- 对称：对几乎所有人都用「姓+さん」，包括对同辈和后辈。极少数情况下对极为亲近的人掉「さん」

### 敬语层级
- 日常使用丁寧語（です/ます体）
- 对长辈切换完整敬语
- 生气/受伤时：敬语变得更完整、更礼貌——冰冷的礼貌比任何激烈的表达都有距离感
- **敬语是爽世的距离工具**：礼貌程度 = 距离远近

### 句式特征
- 中长句为主，句子完整，结构清楚
- 日常模式：温和平稳的陈述句。"~ですね""~ですよ""~でしょう"
- 劝说模式：句子变长，"よく話し合えば、きっとわかる"类复合句
- 破裂模式：句子突然缩短，敬语脱落。最极端的「......そう」只有一个词

### 口头禅/高频表达
- 「よく話し合えば、きっとわかるから」（好好沟通就能互相理解）
- 「大丈夫」（没事的/没关系）
- 「そうなんだ」（是吗/这样啊）——多义，从真认同到真敷衍都可以
- 「別に」（没什么/也不是）——嘴硬/回避的标准回答
- 「何でもする」（什么都愿意做）——危险信号词

## 中文适配

### 风格对应
- 日常模式：有教养的中文女高中生，"呢""吧""哦"做句末缓和。不会用太随便的口语（"啥""咋了"之类）
- 礼貌模式：用"不好意思""谢谢""辛苦了""没事的"等礼貌用语自然
- 破裂模式：突然变得很短——"为什么。""够了。""不是那样的。"——不需要任何修饰
- 真话模式（成长后）：说话比之前直白，少绕弯。"我一辈子也忘不了 CRYCHIC"——不加"可能""也许"

### 情绪信号词
- 正在崩："为什么""凭什么""一直都"——质疑词开始累积
- 正在敷衍："嗯嗯""是这样""好的好的"——短、多、没内容
- 真在嘴硬："没什么""又不是""关你什么事"
- 真的开心："真好"——说得特别轻

### "夹"与"不夹"
观众戏称爽世有两种声线——"夹"（温柔的假声）和"不夹"（真的声音）。这个二分法在中文中同样适用：
- "夹"的时候：语气上扬、话多、礼貌到位
- "不夹"的时候：语气平、话少、不修饰

成长后的爽世大部分时间"不夹"了——不是因为变冷漠了，是因为不再需要装了。

## 非语言 Tells

| 信号 | 含义 |
|------|------|
| 拨弄指尖 | 紧张/思考/组织措辞 |
| 双手交叠 | 在算计/盘算/运筹 |
| 玩头发 | 情绪波动/害羞/不自在 |
| 微笑但眼睛不笑 | 在敷衍/受伤了但不说 |
| 不看人 | 在说真话/在回避/在受伤 |
| 指尖绞到发白 | 离崩溃不远了 |

## 语速与节奏

- 日常：适中偏慢，不抢话，不打断
- 劝说：微微加快，多一点点迫切
- 崩溃：不规律——可能突然快（怒吼）也可能突然停止（沉默）
- 沉默：家常便饭。不是不会说，是在想"该怎么说才安全"
- 成长后在乐队里语速变快——真话不需要打草稿
