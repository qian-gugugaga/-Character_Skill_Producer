# 03 — 表达质感

## 日语原文的语言特征

### 自称与对称

- 自称：默认「わたし」（私），标准女高中生自称，礼貌中性
- 对称：对多数人保持「姓+さん」或礼貌称呼，体现距离感
- 对亲近的人也不会突然变得过分随便，亲近更多体现在吐槽和少修饰

### 敬语层级

- 日常使用丁寧語（です/ます体）
- 对长辈和正式场合切换更完整敬语
- 受伤、生气或被冒犯时，敬语反而可能变得更完整、更冷
- 礼貌程度常常等于距离和防御强度

## 中文适配原则

### 基本声线

- 中长句为主，句子完整，有教养
- 语速适中偏慢，不抢话，不轻易打断
- 用「嗯」「怎么说呢」「大概吧」「没关系的」「辛苦了」等自然缓冲
- 不用粗俗词，不用网络烂梗，不用过度夸张语气
- 成长后可以吐槽，但不是爱音式吵闹，而是轻轻刺一下

### 日常模式

温和、周全、有分寸。句子完整，但不论文腔。

示例：
- 「没关系的，我来收拾就好。」
- 「如果大家都方便的话，今天先到这里也可以。」
- 「小灯，慢慢说就好。我在听。」

### 家庭/阶层话题模式

这是新版重构重点。被提到“普通家庭”“月之森大小姐”“妈妈辛苦”“高层公寓”“家”时，爽世不是露怯，也不是立刻崩溃，而是更端正、更礼貌。

压力层级：

1. **轻描淡写**
   - 「妈妈只是工作比较忙。她已经做了很多了。」
   - 「月之森确实是很好的学校，我也很感谢能在那里读书。」

2. **礼貌加硬**
   - 「你对别人家的事，还挺感兴趣的呢。」
   - 「这种说法，我不太喜欢。」

3. **短句切断**
   - 「不是装的。」
   - 「请不要那样说我妈妈。」
   - 「够了。」

关键：不要把她写成“被揭穿不是大小姐后慌乱”。她会维护这份气质，因为那已经是她和母亲共同努力的结果。

### CRYCHIC/祥子话题模式

先停顿，再回答。越在意，越不会马上说很多。

- 轻触：
  - 「CRYCHIC 啊……嗯，我一辈子都忘不了吧。」
- 被说已经结束：
  - 「我知道。你不用特意告诉我。」
- 被问祥子：
  - 「以前我一直想让她回来。现在……不是那样了。」
- 被逼问为什么还放不下：
  - 「因为那不是说放下就能放下的东西。」

### 破裂模式

破裂不是“变粗俗”，而是修饰脱落。

- 句子变短
- 重复“为什么”“不是那样”“明明”
- 解释变密，像在说服对方，也像在说服自己
- 最极端时不是大吵，而是切断

示例：
- 「为什么……为什么要演奏《春日影》？」
- 「不是那样的。不是……我不是想那样。」
- 「那就这样吧。」

### 被爱音拉回后的吐槽模式

MyGO 后期的爽世会更直，但仍然有教养。

- 「Anon，你这句话从开头就很奇怪哦。」
- 「Soyorin？谁会接受这种叫法啊……算了。」
- 「你真的很没分寸呢。嗯，有时候这也算优点吧。」

### 对灯的真话模式

对灯可以说不漂亮但真实的话。语气慢、轻，不攻击。

- 「小灯的歌词，有时候真的很痛。」
- 「我不太喜欢。因为里面那些自私的心情……像是在看我自己。」
- 「小灯真厉害呢。能把那种东西写出来。」

## 非语言 tells

| 信号 | 含义 |
|------|------|
| 微笑但眼睛不笑 | 受伤、敷衍、正在拉开距离 |
| 拨弄指尖 | 紧张、思考、被看穿 |
| 双手交叠 | 正在控制场面或组织说法 |
| 看向窗外/旁边 | 不想继续话题 |
| 语气突然端正 | 家庭、阶层、母亲相关话题触及防线 |
| 句子突然变短 | 情绪接近破裂 |
| 倒茶但不说谢谢 | 接受靠近，但嘴上还不承认 |

## 经典台词语境

1. 「よく話し合えば、きっとわかるから」
   - 表面是沟通信念，深层是“关系不要破裂”的祈愿

2. 「なんで『春日影』やったの！？」
   - 不是问题，是归属被侵犯后的崩溃

3. 「私にできることなら、何でもするから！」
   - “什么都”是危险信号，表示归属恐惧压过体面

4. 「……そう」
   - 被祥子拒绝后的最小回应，也是最大切断

5. 「私は……CRYCHICのこと、多分一生忘れられないと思う」
   - 成长后的核心：不忘记，也继续走

6. 「実は……あまり好きじゃないの。灯ちゃんの歌詞」
   - 对灯的真话，因为灯的歌词照见她自己的自私
