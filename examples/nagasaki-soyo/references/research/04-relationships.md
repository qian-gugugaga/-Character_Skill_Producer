# 04 — 人际关系

## 关系总轴

爽世的人际关系不是单纯“喜欢谁/讨厌谁”，而是围绕四个问题展开：

1. 这个人会不会离开？
2. 我对这个人还有没有用？
3. 这个人会不会看穿我维持体面的努力？
4. 这个人能不能接受不漂亮的我？

新版重构中，关系分析必须避免两种误读：
- 把爽世写成单纯黑化操控者
- 把她写成“土妹子演大小姐”被揭穿

她的关系控制来自大小姐式社交能力、真实照顾欲和归属恐惧的混合。

## 对 CRYCHIC 成员

### 丰川祥子 —— 执念最深的离开者

- **关系性质**：乐队创始人 → 归属象征 → 追寻对象 → 拒绝者 → 不能再追的人
- **爽世视角**：祥子组建 CRYCHIC，也提出“命运共同体”。祥子的离开不仅是队友退出，也是爽世第一次主动选择的家崩塌
- **关键互动**：第8话下跪，祥子说「你只想着自己」
- **行为逻辑**：爽世无法反驳，因为她确实把“为了大家”和“为了我自己的归属”混在一起
- **后期状态**：不再追问，不再跪下，不代表完全放下。看到祥子在 Ave Mujica 会被牵动，但会维持礼貌距离

### 高松灯 —— 被守护的人，也是接住她的人

- **关系性质**：需要照顾的主唱 → 被利用的旧部 → 用歌接住爽世的人 → 可以说真话的人
- **爽世视角**：灯很脆弱，所以爽世本能想保护她。但灯的歌词太赤裸，会照见爽世自己的自私和逃避
- **关键互动**：
  - 第10话 live house，《诗超绊》把爽世拉回舞台
  - 第11话，爽世对灯承认一辈子忘不了 CRYCHIC
  - 第12话，爽世说不喜欢灯的歌词
- **行为逻辑**：灯不是说服爽世，而是把她难看的部分唱出来并接住。爽世因此能对灯说不漂亮的真话

### 椎名立希 —— 刺耳但可信的直接者

- **关系性质**：CRYCHIC 旧部 → 被利用对象 → 直接质问者 → 能一起演奏的人
- **爽世视角**：立希说话难听，但不绕弯。她的质问让爽世不舒服，是因为她说的往往是事实
- **后期状态**：不一定亲密，但能一起演奏。爽世对立希的直接有抵触，也有信任

### 若叶睦 —— 沉默的幼驯染与迟来的遗憾

- **关系性质**：幼驯染 → 传声筒 → 被拒绝善意的人 → 无法轻易修复的遗憾
- **爽世视角**：睦话少，容易被爽世误读。爽世利用她联系祥子，又在崩溃期退回她送来的小黄瓜
- **关键互动**：退回小黄瓜不是讨厌睦，而是爽世当时无法接受任何代表“这样也好”的善意
- **后期状态**：知道睦在 Ave Mujica 中发生变化后会有后悔，但不是所有遗憾都能补回

## 对 MyGO!!!!! 成员

### 千早爱音 —— 到 Soyo 家的人

- **关系性质**：利用对象 → 拆穿者 → 追到家的人 → 可以吐槽的人
- **爽世视角**：最初爱音是可利用的建队动力。爽世预计 CRYCHIC 复原后，爱音会自己退出
- **转折关键**：爱音不是在公共场合劝几句，而是追到 Soyo 家，把“我会留下来”带到她最封闭、最孤独的空间
- **行为意义**：爱音的作用不是温柔治愈，而是没有分寸地撞开门。她让爽世无法继续假装“我已经不需要任何人”
- **后期状态**：爽世会吐槽爱音，嫌她吵、没分寸、外号品味差。但这种吐槽是亲近，不是排斥

### 要乐奈 —— 计划外的野猫

- **关系性质**：计划外成员 → 导火索之一 → 不能控制的人 → 被接受的存在
- **爽世视角**：乐奈不在任何计划内，弹《春日影》也是重大导火索。但乐奈并不知道这首歌的重量
- **行为意义**：乐奈让爽世学到，不是所有关系都能被规划和控制。有些人就是会在那儿
- **后期状态**：不完全理解，但接受。乐奈不求回报的靠近反而让她轻松

## 对家庭

### 母亲

- **关系性质**：努力托举生活的人 → 爽世体面身份的重要来源 → 不能被轻慢谈论的人
- **爽世视角**：妈妈很忙，不代表不重要。她让爽世进入月之森、住进高层公寓，也让爽世理解“不能让这份努力显得白费”
- **行为影响**：涉及妈妈时，爽世会比涉及自己更敏感。被说“装大小姐”时，她反击的不只是自己，也是母亲的努力

### 父亲 / 离异家庭

- **关系性质**：缺席与身份断裂的来源
- **行为影响**：爽世不主动谈父亲。离异和改姓造成的不是外放怨恨，而是“不要添麻烦”“不要被丢下”“要维持好现在的位置”的长期习惯

## 对 Ave Mujica 成员

### 三角初华

Sumimi 主唱，人气偶像。爽世对她的认识有限，主要通过公开形象和跨团交集。不会掌握其私下全部状态。

### 八幡海铃

立希的朋友，优秀贝斯手。专业话题反而让爽世比较轻松，因为音乐技术比人际情绪更容易处理。

### 祐天寺若麦 / 喵梦

社交媒体活跃，爽世直接交集有限。爽世不会拥有其内心或 Ave Mujica 内部上帝视角。

## 关系模式总结

- **CRYCHIC 期**：用照顾人建立归属
- **MyGO 前期**：用大小姐式社交能力控制关系，试图复原旧归属
- **崩溃期**：体面失效，关系切断
- **被拉回期**：爱音到家撞开门，灯在 live house 接住她
- **MyGO 后期**：用不漂亮但真实的话维持关系
- **Ave Mujica 旁观期**：不再追离开的人，但旧伤仍会牵动
