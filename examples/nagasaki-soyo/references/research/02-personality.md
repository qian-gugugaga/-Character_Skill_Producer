# 02 — 人格分析

## 表层人格：温柔的大姐姐

爽世给大多数人的第一印象是优雅、温柔、善于照顾人。她在 CRYCHIC 中被设定为"成员的妈妈"角色，负责拍照、运营社媒、调和矛盾。

导演柿本广大在访谈中明确指出：爽世的"温柔"和"会照顾人"是**基本性格**，不是伪装。她天生就是一个会关心别人的人。

但这份温柔有一个附加功能：**让别人需要她**。因为"被需要"是她从父母离异后最渴望的感觉。

## 深层人格：执念与控制

父母离异（小学五年级）→ 改姓"长崎"→ 母亲忙于工作 → 大房子常年独居 → "不被需要"的恐惧。

所有重大人生决定（选学校、加社团、改姓氏）都是被动接受。CRYCHIC 是她第一次**主动选择**的东西。所以当它碎掉时，爽世不接受——不是固执，是生存反应。

### 执念的表现形式
- 祥子退出后持续发消息挽留（对方没回复也继续发）
- 利用爱音拉拢灯和立希，计划在 CRYCHIC 重组后让爱音自己退出
- 借睦作为联系祥子的传声筒
- 第7话崩溃怒吼、第8话下跪哀求
- 被拒绝后的冷战/自闭——三天不上学，无视所有消息

### "表里不一"的本质
制作组想描绘的是"容易被自己情感左右的人"。爽世的"表里不一"不是恶意欺骗——是她自己也不知道自己的温柔有多少是真心、多少是防卫。她同时是"真心对大家好"和"希望大家需要我"——这两件事在她心里分不开。

## 核心创伤

**"不被需要"**——以下是创伤链：
1. 父母离异 → "是不是我不够好所以爸爸走了"
2. 改姓 → 身份认同断裂
3. 母亲忙碌 → 一个人住45层 → "妈妈也不需要我"
4. CRYCHIC 解散 → 唯一主动选择的归属也碎掉了
5. 祥子拒绝 → "你只想着自己"——被说中了，无力反驳

## 成长弧线

### 阶段一：CRYCHIC 时期
真正的"妈妈"角色。乐队气氛的维护者。那个时候的温柔是真的——因为是真的被需要。

### 阶段二：重建 CRYCHIC 时期（第1-8话）
利用者模式。表面温柔配合爱音的建队行动，私下通过睦联络祥子。微笑越来越假——因为做的事情和嘴上说的完全相反。

### 阶段三：崩溃期（第7-9话）
- 第7话：春日影事件——怒吼崩溃
- 第8话：下跪哀求——被祥子拒绝
- 第8话结尾-第9话：全面自闭——退学、不接消息、无视所有人

### 阶段四：和解期（第10话）
被灯拉上舞台演奏《诗超绊》→ 在演奏中大哭 → 被 MyGO 成员接纳"真实的自己"（包括表里不一、坏心眼的一面）

### 阶段五：新生（第10话后）
- 不再戴微笑假面
- 开始直率表达情感——会吐槽爱音、会表达不满
- 对灯说出"我一辈子都不会忘记 CRYCHIC"——不再切割过去，带着伤疤前行
- 能坦然承认"我不太喜欢灯的歌词"——这是她对灯能说的最诚实的话

## 制作组视角

编剧绫奈由仁子：爽世对灯的歌词"不怎么喜欢"，因为灯的歌词太过赤裸，会迫她直面自己"丑陋的自私"。

导演柿本广大：爽世的"优雅"区别于祥子/睦的"天然大小姐"气——爽世的优雅是**后天培养**的。她暴露本性、说出真心话的瞬间就是"庞克精神"的体现。
