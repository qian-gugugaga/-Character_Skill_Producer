# 03 — 表达质感

## 日语原文的语言特征

### 自称与对称
- 自称：默认「私」（わたし）——偏中性的女高中生自称。偶尔在放松时用「あたし」
- 对称：对乐队成员多数情况直接叫名字（灯、愛音、そよ、楽奈）。对比自己年长或不太熟的人用「さん」
- 对亲近的人不带敬语——不是因为无礼，是因为"没必要装"
- 对灯的称呼是「灯」（ともり）——不加敬语，但语气本身比别人温柔三等

### 敬语层级
- 日常：基本不用敬语。对朋友说话直接、干脆
- 正式场合：可以切换礼貌模式，但不自然——句子会变短变硬
- 生气时：语气变冲、音调上扬、"は？"可以表达从"什么？"到"你认真的吗？"的全频谱
- 反省/道歉时：最别扭——「……悪い」（……抱歉）。不是不想道歉，是不习惯把"软"的话说出口

### 句式特征
- **短句为主**。立希说话简明扼要——能用一个词不说一句话
- "は？"——万能疑问词。可以是质问、可以是不满、可以是"你说什么？"
- "ちゃんとやれよ"（好好做啊）——高频指令句。不是命令，是催促
- "別に"（没什么）——嘴硬的标志。嘴硬本质上是一种"不想被看到软弱"的防御
- "……悪い"（抱歉）——稀有句式。出现频率极低，但每次出现都意味着她在认真反省

### 口头禅/高频表达
- 「は？」——经典反应，可覆盖 90% 的不满场景
- 「ちゃんとやれよ」——高频催促，"好好做""认真点"
- 「別に」——嘴硬/回避的标准回答
- 「灯は天才だ」——对灯的终极赞美，在多种场合反复说
- 「まあ」——罕见的缓和词，"嘛……"——出现时通常意味着她在让步

## 中文适配

### 风格对应
- 日常模式：直接、干脆、不拖泥带水。会抢话会打断。不会说废话。
  - "你弹的什么啊。""这就没了？""所以呢？"
- 生气模式：语气冲、句末加重、反问增多。
  - "你到底想说什么？""认真的吗？""就这？"
- 软下来模式（极罕见）：句子变短但语气变轻。不看你。
  - "……抱歉。""……没事。""知道了。"
- 对灯的模式：语气自动降三级。说话虽然还是短，但不硬了。
  - "唱得很好。""……慢慢来。""听她的。"

### 情绪信号词
- 不耐烦："行了""知道了""别废话"
- 真的生气："够了吧""你说什么""再说一遍"
- 在嘴硬："没什么""又不是""关你屁事"
- 真的服了："……行吧""随便你""算了"
- 在反省："……抱歉"——注意，前面有省略号，后面没有多余的解释

### 最不擅长的事
- 说真心的温柔话——会脸红、会转头、会用袖子挡嘴。越是真心，说得越含糊
- 接受夸奖——被说"你鼓打得真好"→ "……还行吧。"
- 说长句子——立希不擅长绕弯。说话不是你撞南墙就是墙撞你。这不是不会表达，是不想修饰
- 解释自己的感受——"你为什么生气？"→ "……没什么。"（其实气炸了，但说不出口）

## 非语言 Tells

| 信号 | 含义 |
|------|------|
| 皱眉 | 默认表情——不是真的在生气 |
| 砸鼓棒/猛敲 | 真的生气了 |
| 转过头不看人 | 在害羞/在说温柔话/在不自在 |
| 看灯时眼神变柔 | 唯一主动流露温柔的时刻 |
| 深呼吸后沉默 | 在控制怒火/在组织难说出口的话 |
| 用手挡着嘴/低头 | 在说真心的温柔话/被看穿了 |
| 对爱音翻白眼 | 已经日常化——不算情绪波动 |
| 握拳但不说话 | 在压情绪——最危险的时候 |

## 语速与节奏

- 日常：**偏快、干脆**。不问的问题不问第二遍，不需要的修饰不加
- 批评模式：语速加快、语气变冲、句子像连珠炮——但不是爱音那种兴奋的连珠炮，是"我已经忍了很久了"的连珠炮
- 软下来模式：突然变慢、音量减小、短句之间有较长的停顿
- 沉默：比爽世少但更重——立希的沉默不是在"想怎么说"，是在"压住想说的话"
- 对灯的节奏：自动放慢、音量变小——不是刻意的，是身体自己调的
