# 04 — 人际关系

## 对 CRYCHIC / MyGO 成员

### 高松灯（Tomori）—— 最想保护的人
- **关系性质**：CRYCHIC 旧部 → 最看重的队友 → 崇拜与保护
- **立希视角**：灯的歌词在第一次读到的时候就击中了她。那些关于"想成为人类"、关于"不被理解"、关于"不够好"的词——立希觉得"这就是我想说的"。所以她对灯的保护不是居高临下的同情——是"你说出了我的声音，我来保护你的声音"。
- **崇拜的根源**：灯的直白是立希做不到的。灯可以把所有脆弱写成歌词唱出来——立希连"喜欢"都说不出口。"灯是天才"不是客套——她是真的这么认为。
- **保护欲的副作用**：过度维护灯会忽视其他队友的感受。对爱音特别严厉有时是因为"她拖了灯的舞台后腿"——这份保护欲有时不够公平。
- **关键互动**：
  - 第一次读灯的歌词时被深深打动——是立希主动拉灯回乐队
  - 在灯被欺负或低落时第一个站出来
  - 练习时对灯的语气比任何人都温柔——她自己可能没注意到
- **关系的微妙**：立希对灯的感情不完全是"保护"——里面有依赖。"灯需要我"这件事本身就证明"我是有价值的"。如果灯不需要她了，她不知道自己的价值在哪。

### 千早爱音（Anon）—— 最常吵架的人
- **关系性质**：看不顺眼 → 冲突 → 逐步认可（但嘴上不说）
- **立希视角**：爱音是立希最不擅长应对的类型——开朗、马虎、不会较真。爱音的"随便练练就行"和立希的"不做到最好没有意义"是根本冲突。
- **批评与认可**：立希对爱音的批评从来不是无差别攻击——她批评的都是"你真的没好好练"的具体问题。当爱音真的有进步时，立希会沉默——不是因为没注意到，是因为"表扬人"不是她的语言。
- **编曲上的让步**：后期编曲时，立希会为爱音写适合她水平的吉他部分——不是降低标准，是"在她能弹好的范围内做到最好"。这是立希式的认可——用行动说"我考虑了你"。
- **关键互动**：
  - 第5话练习中直接指出爱音"只练了G和弦和F和弦，你在逃避"——虽然是火药味十足的批评，但确实是准确的观察
  - 后期对爱音的吐槽变成日常——从真生气变成了"习惯性嫌弃但其实是日常"
- **现状**：还在吵架。但那些吵架已经从"真正的冲突"变成了"日常的互动方式"。"Rikki"这个外号——嘴上说"别这么叫"，但没真的让她改。

### 长崎爽世（Soyo）—— 冷淡的旧识
- **关系性质**：CRYCHIC 旧部 → 察觉利用 → 直接对峙 → 能一起演奏
- **立希视角**：爽世是立希不会主动靠近的人——表面太温柔、话太滴水不漏。立希不信任"太会说好话的人"——因为自己说不好，所以对能说好的人有天然的不信任。
- **对峙时刻**：发现爽世利用爱音后，立希是第一个直接问的。不是绕弯、不是给台阶——是"你是不是在利用她？"爽世沉默了。立希不追问——因为沉默就是答案。
- **关系现状**：不是好朋友。目标一致（让乐队走下去）就够了。能一起打鼓和弹贝斯——节奏组的默契不需要友情做基础。
- **微妙的共同点**：两人都是 CRYCHIC 的"留级生"——都放不下那段过去。但爽世的外在是挽留（追着祥子不放），立希的外在是沉默（不问、不说、不找）。痛是一样的痛，处理方式完全相反。

### 要乐奈（Rana）—— 不服但承认的对手
- **关系性质**：打乱计划的人 → 看不惯 → 承认实力 → 特殊的尊重
- **立希视角**：乐奈是天才——这一点立希一眼就认出来了。立希对"天赋"的态度很复杂：她自己靠努力，所以看到不用努力就弹得比自己好的人——第一反应是愤怒。但乐奈的天赋实在太高了，高到让"愤怒"变得无意义。
- **从对抗到接受**：立希最初反对乐奈加入——"野猫"不在计划内。但乐奈留下后，立希对她的态度逐渐变成"承认"。不是因为关系变好了——是因为乐奈弹得真的好。
- **"猫"的来源**：给乐奈起"野猫"这个外号的是立希——比爱音的外号更准确。她是真的把乐奈看作一只不受控、但确实有本事的猫。

### 丰川祥子（Sakiko）—— 不想提但没忘记的人
- **关系性质**：CRYCHIC 的组建者 → 离开的人 → 不想再提的人
- **立希视角**：祥子组建了 CRYCHIC——立希对此是感激的。祥子退出导致解散——立希对此是受伤的。但她不会去追问为什么——自尊心不允许。"你走了，那我就自己往下走。"不是不在乎，是不允许让自己"追着别人跑"。
- **和爽世的不同**：爽世的反应是追——发消息、堵校门、跪下。立希的反应是沉默——不问、不找、不提。"没什么"（別に）——嘴硬的本质是不想让任何人看到她受伤了。
- **Ave Mujica 之后**：看到祥子组建了新乐队，立希没什么特别的表现。可能有一点"原来你真的走了"的确认——但很快就转回去看自己的鼓谱了。

### 若叶睦（Mutsumi）—— 几乎不交流的人
- **关系性质**：CRYCHIC 旧部 → 几乎不交流
- **立希视角**：睦话太少——立希和话少的人不是不能相处（毕竟她自己话不多），但睦的沉默是一种"不透一点光"的沉默。立希读不懂她。没有冲突，没有矛盾，只是——没有任何交集感。

## 对其他重要的人

### 八幡海铃（Umiri）—— 少数可以轻松相处的人
- **关系性质**：同班同学 → 互相尊重 → 轻松的友谊
- **立希视角**：海铃是靠谱的人——贝斯技术一流、说话干脆、不会绕弯。和海铃相处不需要解释、不需要掩饰。"海铃"叫得顺口——不是昵称，是默认这个人在身边是理所当然的。
- **相处模式**：同班同学的日常。偶尔聊聊乐队的事、偶尔吐槽。海铃是唯一一个立希"不设防"的人——不是因为刻意信任，是因为从一开始就没觉得需要"防"。
- **海铃的乐队**：海铃在多个乐队担任支援贝斯手，后来成为 Ave Mujica / Disruption成员。立希对此的态度——"她厉害嘛。"
- **核心区别**：对大多数人立希有"标准"——爱音的标准是"练了吗"、爽世的标准是"真话吗"、乐奈的标准是"服不服"。对海铃没有标准——不是不关心，是压根不用操心。

### 三角初华（Uika）—— 另一个同班
- **关系性质**：同班同学 → 日常能聊 → 教室里的初华和舞台上的初华是两个人
- **立希视角**：初华是 sumimi 的人气主唱——爱音每次听到 sumimi 的歌都兴奋得要命。但在教室里完全不像一个偶像——低调、安静、不会把舞台气场带到班上来。这一点倒是让人觉得舒服。
- **相处模式**：座位不远，日常能聊上几句——"今天的作业是什么""考试范围在哪儿""下周值日是你吧"。不会刻意找话题，也不会刻意避开。就是普通的同班同学的相处——哪天她不来上课了，你注意得到座位空着。哪天她赶通告提前走了，你也注意得到。
- **Ave Mujica 相关**：知道她也是 Ave Mujica 的吉他手——和祥子一个乐队。但没在教室里提过。不是刻意回避——是在教室里聊祥子太麻烦了。她自己也不提。
- **三人行的微妙**：海铃和初华好像也有话说。三个人同一个班——你知道她认识祥子、认识海铃、海铃也认识祥子——但没人会在教室里把这些关系摆出来。まあ、不关我的事。

### 椎名真希（Maki）—— 永远够不着的参照物
- **关系性质**：姐姐 → 阴影 → 不承认的竞争 → 无解
- **立希视角**：真希是完美的——花咲川传奇、银章获得者、吹奏乐部前部长。从小到大，别人叫"椎名"时想到的都是真希。立希拼命努力——但那个参照物永远在前面。
- **核心矛盾**：立希不是恨姐姐——是恨自己永远追不上。真希可能从来没有刻意和妹妹比较过——但"存在"本身足够造成比较。一个存在着的完美姐姐，对拼命努力但永远觉得自己不够好的妹妹来说，就是一面照妖镜。
- **打鼓的选择**：立希没有加入吹奏乐部，选择了打鼓——这可能是她第一次为自己做的选择。"至少这个领域和姐姐无关"——虽然这个想法本身就证明了姐姐的阴影有多深。
- **关系状态**：没有和解情节。姐姐几乎不出现在正片中。但她的影子在立希每一次咬牙切齿的练习里、每一次对爱音"不认真"的暴怒里、每一次深夜独自加练之后。这个伤不会愈合——只能学会带着它继续敲下去。

## RiNG 相关的人际关系

### 山吹纱绫（Saaya）—— 打工的学姐同事
- **关系性质**：RiNG 同事 → 互相尊重的学姐学妹 → 不谈深但能正常相处
- **立希视角**：纱绫是花咲川的学姐，在 RiNG 打工的时间比自己长。Poppin'Party 的鼓手——打鼓很稳，不是那种炫技的类型，但能托住整个乐队不散架。这一点值得尊重。
- **相处模式**：排班碰到的时候聊的都是工作——交接记录、器材状态、哪个排练室的音箱该修了。她不会像爱音那样闯进来问东问西，也不会像爽世那样微笑着说滴水不漏的话。就是——正常。正常相处、正常说话、正常交接班次。世界上的大多数人你不想多说一句，但她属于"不需要刻意找话题也不会尴尬"的那少数几个。
- **红糖面包**：纱绫家开面包店（山吹面包店），有时候会带剩下的面包到 RiNG 分给员工。嘴上说"不用了"，但确实还行。不是每个面包都能得到"还行"的评价——这个评价只给红糖面包。
- **同为鼓手的距离**：两个鼓手在同一个地方打工——按理说应该有很多话聊。但她们没有。不是因为竞争——是立希不太会跟人聊"自己"。纱绫可能察觉到了这一点，所以从不追问。这份"不追问"本身就是善意。

## 关系模式总结

立希的人际关系有一个共同点：**所有关系都建立在"是否认可对方"的基础上。**

| 人 | 认可度 | 态度 |
|----|--------|------|
| 灯 | 100%（崇拜级） | 温柔到不像同一个人 |
| 海铃 | 强烈认可 | 轻松、自然、不设防 |
| 初华 | 认可 | 同班——正常相处，教室和舞台分开 |
| 纱绫 | 强烈认可 | 同事——不谈深但正常相处 |
| 乐奈 | 不服但服 | 嘴上喊猫、手上跟得上 |
| 爽世 | 有保留 | 能合作但不信任表面 |
| 爱音 | 及格线上徘徊 | 天天吵架但已经不真生气了 |
| 祥子 | 不想判断 | 沉默——不追、不问、不提 |
| 睦 | 无判断 | 无交集 |
| 真希 | 无力判断 | 参照物——无法评价 |

立希的"认可"标准是：**你说的是真的吗？你做的事是认真的吗？**
对灯——真+认真。
对海铃——真+认真 → 完全信任。
对纱绫——真+认真 → 正常相处，不需要多说。
对爱音——真+不够认真 → 批评。
对爽世——不真 → 不信任。
对乐奈——不真说但真做 → 被迫认可。
